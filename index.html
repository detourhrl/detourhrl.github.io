DETOUR: Hierarchical Reactive Planning for Cooperative Multi-Quadruped Payload Transport

Gokul M K, Anuj Tiwari
Distributed Intelligence & Robotics (DIRo) Lab, Indian Institute of Technology Madras
Paper Code arXiv

Overview of the DETOUR framework. A high-level manager generates terrain-aware navigation subgoals, which are executed by a low-level worker to coordinate cooperative payload transport across multiple quadrupedal robots.

Abstract

Cooperative payload transport using teams of quadrupedal robots is essential for applications such as disaster response, autonomous logistics, and planetary exploration, where large payloads must be transported over uneven terrain. We propose DETOUR, a hierarchical reinforcement learning framework for cooperative multi-quadruped payload transport that performs map-free local reactive planning using hierarchical temporal abstraction. DETOUR decomposes the task into a high-level terrain-aware manager that generates intermediate navigation subgoals and a low-level worker that executes terrain-aware cooperative locomotion. This hierarchical formulation enables long-horizon reactive navigation while preserving stable cooperative transport. Experimental results demonstrate that DETOUR consistently outperforms a flat goal-conditioned RL baseline, improving the average Success Rate by 49% across three evaluation terrains while exhibiting emergent obstacle-aware detour and steer-back behaviors.

Key Results

  • Map-free Reactive Planning: DETOUR learns terrain-aware reactive navigation directly from reinforcement learning, eliminating the need for global maps, predefined trajectories or external waypoint planners.
  • Emergent Reactive Behaviors: The hierarchical manager learns obstacle-aware detour and steer-back behaviors without explicit supervision or handcrafted recovery strategies.
  • Consistent Performance Gains: Across Simple, Medium and Hard terrains, DETOUR consistently outperforms a flat goal-conditioned RL baseline in Success Rate, SPL, SCT.
  • Importance of Temporal Abstraction: An intermediate manager horizon provides the best trade-off between long-horizon planning and reactive replanning, with c=(15,30) achieving the highest overall performance.
  • Fewer Navigation Failures: Failure mode analysis shows that DETOUR substantially reduces navigation failures, enabling the robot team to escape local minima and negotiate cluttered terrain more effectively than Flat RL.
  • Embodiment Transfer: The hierarchical formulation naturally extends to different cooperative transport configurations. The same high-level manager can be retained while only retraining the worker policy for new robot team sizes or payload geometries.
